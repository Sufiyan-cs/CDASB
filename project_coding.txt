# SOFTWARE REQUIREMENTS SPECIFICATION (SRS)

## Project Title

**Conflict-Driven Autonomous System Builder (CDASB) – Interactive Version**

---

# 1. INTRODUCTION

## 1.1 Purpose

This system is designed to:

* Accept prompts OR documents (.pdf / .docx)
* Plan solutions using AI agents
* Use structured conflict to refine decisions
* Require user approval before execution
* Iteratively build, test, and refine systems
* Provide deployment or local execution options

---

# 2. UPDATED CORE WORKFLOW (KEY CHANGE)

## 2.1 End-to-End Flow

```text
User Input (Prompt / PDF / DOCX)
        ↓
Planning Phase (Agents)
        ↓
Conflict Phase (Debate on Plan)
        ↓
Final Plan + Implementation Strategy
        ↓
User Approval (Proceed Button)
        ↓
Execution Begins
        ↓
Conflict → Coding → Conflict → Testing → Conflict
        ↓
Completion Notification
        ↓
User Choice:
   [Deploy] or [Run Locally]
        ↓
Final Testing → Conflict → Output Delivery
```
